Projenin doğru çalışabilmesi için çalıştırılacak bilgisayarda son sürüm python ve visual studio code kurulu olmalıdır.

Visual Studio Code içerisindeki extensions kısmından python eklentileri yüklenmelidir.

Projenin gönderildiği zip dosyasının içinde PLY kütüphanesi için gerekli olan yacc ve lex dosyaları bulunmaktadır. Bu kütüphane için başka işlem yapılmasına gereksinim yoktur.

cmd'ye yazılacak:
pip3 install PythonTurtle
pip3 install tk
pip3 install customtkinter

komutları ile gui için gereken kütüphaneler bilgisayara eklenecektir.

Zip dosyası klasöre ayıklanmalıdır, Ardından Visual Studio Code açılmalı ve file -> open folder ile codes_1 klasörü seçilmelidir.

Visual Code içerisindeki soldaki panelden '1.py' adlı python dosyası açılıp run edilmelidir.
